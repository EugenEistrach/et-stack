# create-et-app

A CLI tool to create new applications using the et-stack boilerplate.

## Usage

```bash
# Using pnpm (recommended)
pnpm create et-app my-app

# Using npm
npm create et-app my-app

# Using yarn
yarn create et-app my-app
```
